Chat Widget Integration Examples

This page demonstrates different ways to integrate the secure chat widget.

Basic Security (Domain Validation)

window.ChatWidget.boot({
  app_id: "demo_app",
  organizationId: "demo_org_123",
  apiBaseUrl: "http://localhost:3000",
  email: "demo@example.com",
  name: "Demo User",
  user_id: "demo_user_123",
  onSecurityError: (error, code) => {
    console.error("Security error:", error, code);
  }
});

Enhanced Security (JWT Authentication)

// Generate JWT server-side, then:
window.ChatWidget.boot({
  app_id: "demo_app", 
  organizationId: "demo_org_123",
  userJWT: "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...", // Your JWT
  apiBaseUrl: "http://localhost:3000",
  email: "secure@example.com",
  name: "Secure User",
  user_id: "secure_user_123"
});
